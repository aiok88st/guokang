  • 就医转诊服务体系

经过12年的就医服务经验积累，国康形成了行业最强大的，满意度最高的就医转诊体系，该体系由专业分诊体系、专家预约体系、专业陪诊体系、大病专案体系、建立档案系统组成，

涵盖全国3万多名各学科、各病种的专家，为客户提供防误诊误治、解决看病难、快速找名医、同一专家连续性看诊、提高就诊治疗效率的服务。国康就医转诊服务体系

已经服务了近千家大中型企业，服务总量超过了100多万人次，误诊误治率为零（国内平均误诊误治率超过了30%），国康就医转诊服务体系深得客户信赖。

  • 专业陪诊体系

  • 12年的服务经验形成对各三甲医院的环境，科室，就医注意事项，专家擅长等知识，客户通过微信预约，电话预约等方式预约专业陪诊服务，陪诊体系可提供安排号源。诊前沟通，就诊陪护，诊中关怀，取送检查报告，诊后关怀等服务，缓解客户在医院迷茫的就医环境。

  • 专业陪诊体系

  • 12年的服务经验形成对各三甲医院的环境，科室，就医注意事项，专家擅长等知识，客户通过微信预约，电话预约等方式预约专业陪诊服务，陪诊体系可提供安排号源。诊前沟通，就诊陪护，诊中关怀，取送检查报告，诊后关怀等服务，缓解客户在医院迷茫的就医环境。

  • 就医转诊服务体系

专业分诊体系是一套有关如何精准就医，高效看病的医学专业知识体系。

国康总结了12年就医服务的最佳实践，掌握了各三甲医院的特长，各名医的擅长以及各种重大疾病的国内外临床治疗路径。

专业分诊体系包括一般疾病分诊和重大疾病分诊。

一般疾病分诊

国康全科医生针对客户病情进行精确分诊，避免看错科室、浪费时间以及造成不必要的费用损失

重大疾病分诊

针对比较复杂的病情，安排三甲医院的专科医生对客户病情进行精确分诊，避免误诊误治以及为治疗时间争分夺秒，提供高效率的看诊。

分诊知识库

经过12年的就医服务经验以及病症案例形成了海量的分诊知识库，该知识库分诊词条超过百万条。

  • 专家预约体系

国康有一套完整的服务预约体系，涵盖550家以上三甲医院3万余名顶级专家为客户提供名医绿色通道、知名专家上门服务、

推荐权威专家看诊、知名专家健康讲座服务、重大疾病等服务，为客户提供精准看诊，解决了百万余名客户看病难、环境差、

找不到名医、不能同一专家连续看诊以及健康知识匮乏等痛苦。

  • 名医绿色通道

  • 3万名以上各学科、各病种的顶级专家为客户开通绿色就医通道，帮助客户获得更多更快的就医机会。

  • 推荐对症权威专家

  • 全国三甲医院3万名以上的顶级专家库，可根据客户病情快速匹配最适合、最权威的专家。

  • 国康保健办名医一对一看诊

  • 权威专家在国康保健办坐诊，让客户享有在舒适优雅的环境下与名医深度沟通病情，沟通时长保证在30分钟以上，从而告别2分钟内看诊模式。

  • 专业陪诊体系

12年的服务经验形成对各三甲医院的环境、科室、就医注意事项、专家擅长等知识，客户通过微信预约、电话预约等方式预约专业陪诊服务，陪诊

诊体系可提供安排号源、诊前沟通、就诊陪护、诊中关怀、取送检查报告、诊后关怀等服务，缓解客户在医院迷茫的就医处境。在康复期间国康安排

客户与看诊医生进行一次电话沟通，全面追踪诊后情况，改善客户以及家属的就医体验。

  • 健康助理陪同就医

  • 27个省市550余家三甲医院3000余名护士以及450余名健康助理对各三甲医院就医流程熟悉，为客户提供全程陪同看诊服务，协助安排号源、诊前沟通、陪同检查、取药划价、解决在就医过程中遇到的各种问题，让客户省心更省力。

  • 全科医生陪同就医

  • 在医院10年以上临床经验的主治医师在医院陪同看诊，负责与主诊医师进行病情以及治疗方案的沟通，提高就诊治疗效率。

  • 大病专案体系

  • 国康整合了550余家三甲医院医疗资源，为客户提供安排名医会诊、快速响应机制，已完成超过10万例大病专案服务。汇集了各种疾病的治疗方案以及多种康复方案。通过连接国外一流的肿瘤专家，形成了国内外治疗肿瘤疾病的知识库体系。大病专案体系由权威专家团队与大病专案小组组成。

  • 权威专家团队会诊

    全国各学科擅长重疾的权威专家团队，针对客户病情做出评估并提供最佳治疗方案。

    大病专案小组

    由健康管家、大病专员、全科医生、项目经理等成员建立的快速响应专案小组。

    健康管家

    7x24小时在线随时接听客户需求，与客户保持持续沟通，积极响应客户并安排服务。

    大病专员

    处理客户在医院的一切医疗事务，包括联系专家、安排床位、协调手术时间、取药划价等。

  • 全科医生

    与主诊医生保持持续沟通联系，将客户病情以及需求和主诊医生准确传达，并给客户做出病情相关解答工作以及术后康复指导。

  • 大病专案体系

  • 国康电子健康档案已建立百万余人次就医案例，记录了从发病到治疗的过程，包括各病种的治疗方案、专家推荐、药品使用、治疗费用等

  • 就医历史记录

    建立了全国550余家医院内、外、妇、儿、五官、检验、放射等科室的就医记录，同时根据个人就医形成数据，分析个人身体健康指标管控疾病的复发风险，提前做健康管理计划

  • 大病治疗案例

    已有肿瘤科、心血管内科、心胸外科、神经科、血液科、泌尿外科、肾内科、骨科、肝胆外科、耳鼻喉科、眼科等重大疾病的治疗案例
